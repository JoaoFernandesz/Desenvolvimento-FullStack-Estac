Objetivo

Programa que cria uma entrada de dados de notas, faz uma leitura deles, transforma esses dados em um histograma e por fim realiza uma nuvem de palavras em um arquivo PNG



Análise e Conclusão

A importância de manipular é necessária para realização de apis e leitura de dados.

A importância de visualizar dados em histogramas e em outras representações gráficas se dá a análise de dados que compõe a ciência de dados

Uma disposição gráfica dos números por extenso de acordo com a quantidade da amostra desse número.
